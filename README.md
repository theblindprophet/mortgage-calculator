# Mortgage Calulator
Powered by [https://nodejs.org/](NodeJS) and [https://electronjs.org](ElectronJs).

## Install

`git clone https://github.com/theblindprophet/mortgage-calculator.git`

`cd mortgage-calculator`

`npm install`

## Run

`npm run start`
